# hello git

## git command summary

- clone : 원격 저장소 복사
- add : 스테이지 영역에 작업 파일 추가
- commit : 세이브, 스테이지 영역의 파일을 가지고 커밋을 만들 수 있다
- push : 원격 저장소에 커밋을 업로드한다

## 파일에 내용 추가하기

- 이걸로 branch를 알아보겠습니다

## 브랜치 하기!

- branch : 기존 내용을 유지한 채 새로운 내용을 추가할 때 분기를 만든다
- check-out : 특정 브랜치(혹은 커밋)로 돌아가는 것
- check_out on sourcetree : 브랜치 더블클릭
